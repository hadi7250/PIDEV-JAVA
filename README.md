📚 EduPlatform
A desktop-based educational platform built with JavaFX, designed to connect students, teachers, and administrators in one unified environment — similar to Blackboard.

📖 Description
EduPlatform is a Java desktop application that provides a centralized space for managing educational content and activities. Students can access their lessons, take quizzes, and view their evaluation schedules, while teachers can upload and manage content. Administrators oversee the entire platform and manage users.

👥 User Roles

Student — Access lessons, take quizzes, view evaluation schedules and results
Teacher — Upload lessons, create quizzes, schedule evaluations
Admin — Manage users, monitor platform activity, configure settings


✨ Features

📂 Lesson Browser — Students can browse and access their course materials
📝 Quizzes — Interactive quizzes created and managed by teachers
📅 Evaluation Schedule — View upcoming exams and evaluation dates
📊 Evaluations — Take evaluations and view results
🔐 Role-based Access — Different dashboards and permissions per user role
👤 User Management — Admin panel to manage students and teachers


🛠️ Technologies Used

Technology                  Purpose
